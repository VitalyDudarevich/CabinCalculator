# Калькулятор - Исправление проблем с отображением

## Текущие проблемы
1. На iPhone:
   - Некоторый текст невидим (вероятно, белый текст на белом фоне)
   - Требуется явное указание цвета текста (#000)

2. В Chrome:
   - Появляется черная полоса справа
   - Проблемы с горизонтальной прокруткой

## Требования к исправлениям

### Исправление проблем с текстом на iPhone
- Добавить явные цвета текста для всех текстовых элементов
- Обеспечить контрастность текста на всех фонах
- Проверить все компоненты на наличие неявных цветов текста

### Исправление проблем с отображением в Chrome
- Устранить появление черной полосы справа
- Контролировать переполнение контента (overflow)
- Установить правильные значения максимальной ширины
- Оптимизировать горизонтальную прокрутку

### Общие требования
- Обеспечить кроссбраузерную совместимость
- Поддерживать адаптивный дизайн
- Следовать лучшим практикам CSS
- Исправить все линтер-ошибки

## Технические детали
- Использовать CSS-свойства для контроля overflow
- Применить maxWidth: '100%' где необходимо
- Добавить явные цвета текста (#000)
- Проверить и исправить типы TypeScript

## Критерии приемки
1. Текст должен быть четко виден на iPhone
2. Отсутствие черной полосы в Chrome
3. Отсутствие горизонтальной прокрутки
4. Все линтер-ошибки исправлены
5. Успешное тестирование на всех основных браузерах
